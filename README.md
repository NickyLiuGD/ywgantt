# 扬维公司甘特图应用
📁 项目文件结构
====================

gantt-chart/  
├── index.html              # 主HTML文件  
├── css/  
│   └── gantt.css          # 甘特图样式  
├── js/  
│   ├── utils.js           # 工具函数（日期处理等）  
│   ├── gantt-chart.js     # 甘特图核心类  
│   └── app.js             # 应用初始化和事件绑定  
└── README.md              # 项目说明文档

====================
部署到 GitHub Pages 步骤：
====================

1. 创建仓库并上传所有文件
2. 仓库设置 → Pages → 选择 main 分支
3. 访问 https://{your-username}.github.io/ywgantt/
   即：https://nickyliugd.github.io/ywgantt/
